# Calculadora de Descarboxilación de Cannabis

Calculadora gratuita para calcular la **temperatura y el tiempo de descarboxilación** del cannabis medicinal. Convierte los cannabinoides ácidos (THCA, CBDA) en su forma activa (THC, CBD) de forma controlada, usando cinética de primer orden y la ecuación de Arrhenius.

Herramienta técnica para la elaboración de aceites y extractos de cannabis medicinal.

## ¿Qué hace esta calculadora?

- Calcula la **combinación de temperatura y tiempo** según el material y el porcentaje de conversión buscado.
- Distingue entre **flor y extracto**.
- Convierte **THCA → THC** y **CBDA → CBD**.
- Indica cuándo el riesgo de degradación a CBN empieza a ser relevante.
- Devuelve el resultado en mg/g, listo para la formulación del aceite.
